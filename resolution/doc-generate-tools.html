文档生成工具

文档生成工具

工欲善其事必先利其器！文档生成工具能通过代码生成漂亮的在线 API 文档，帮助开发者快速构建文档，节省效率和维护的成本。

  • docsify.js.org - 一个 JavaScript 编写的文档站点生成器。
  • TypeDoc - TypeScript 项目的文档生成器。
  • Swagger - 使用Swagger开源和专业工具集简化用户，团队和企业的API开发。
  • Spring REST Docs - Spring REST 文档生成工具.
  • MkDocs - 一个快速简单静态站点生成器。
  • Doxygen - Doxygen 是一种开源跨平台的，以类似JavaDoc风格描述的文档系统，完全支持C、C++、Java、Objective-C 和 IDL语言，部分支持 PHP、C#。注释的语法与 Qt-Doc、KDoc 和 JavaDoc兼容。Doxygen可以从一套归档源文件开始，生成 HTML 格式的在线类浏览器，或离线的LATEX、RTF参考手册。
